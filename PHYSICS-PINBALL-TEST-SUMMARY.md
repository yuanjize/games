## 游戏说明

### 如何开始游戏
1. 在浏览器中打开 http://localhost:8888/physics-pinball/
2. 点击 "发射球" 按钮或按空格键
3. 使用左右箭头键控制挡板
4. 击中反弹器获得分数

### 控制方式
| 按键 | 操作 |
|------|------|
| ← / → | 移动挡板 |
| 空格 | 发射球 |
| R | 重新开始 |
| P / ESC | 暂停 |

### 游戏元素
| 元素 | 分数 |
|------|------|
| 反弹器 | +100 |
| 连击倍率 | 最高 4x |
