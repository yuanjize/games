# 水果2048 🍎🍌🍊🍇🍓🍉

一个美味的水果主题2048变体游戏，使用纯原生JavaScript、HTML和CSS开发。

## 🎮 游戏简介

水果2048是一款基于经典2048游戏的创意变体，将数字替换为美味的水果！通过合并相同的水果，创造更高级的水果，最终目标是获得西瓜！游戏支持键盘和手机触摸操作，适合所有年龄段的玩家。

## 🚀 快速开始

1. 下载游戏文件到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

**无需安装任何依赖** - 游戏使用纯原生技术栈开发。

## 🎯 游戏规则

1. **目标**：合并水果直到获得西瓜（🍉）
2. **操作**：使用键盘方向键或手机滑动来控制水果移动
3. **合并**：当两个相同的水果在移动方向上相邻时，它们会合并成下一个等级的水果
4. **得分**：每次合并都会获得分数，合并的水果等级越高，分数越多
5. **游戏结束**：当棋盘填满且无法进行任何移动时，游戏结束

## 🍎 水果合并顺序

```
苹果（🍎） + 苹果（🍎） = 香蕉（🍌）
香蕉（🍌） + 香蕉（🍌） = 橙子（🍊）
橙子（🍊） + 橙子（🍊） = 葡萄（🍇）
葡萄（🍇） + 葡萄（🍇） = 草莓（🍓）
草莓（🍓） + 草莓（🍓） = 西瓜（🍉）🎉
```

## ⌨️ 操作说明

### 电脑（键盘控制）
- **↑ / W**：向上移动
- **↓ / S**：向下移动
- **← / A**：向左移动
- **→ / D**：向右移动
- **R**：重新开始游戏

### 手机（触摸控制）
- **滑动**：在游戏区域向任意方向滑动
- **控制按钮**：使用屏幕下方的方向按钮
- **点击**：点击游戏控制按钮进行操作

## 🎯 游戏技巧

1. **保持角落**：尽量保持最大水果在角落位置
2. **规划移动**：提前规划移动顺序，避免棋盘过早填满
3. **优先合并**：优先合并低等级水果以腾出空间
4. **观察预览**：注意观察下一个即将出现的水果
5. **利用空位**：巧妙利用空位进行合并
