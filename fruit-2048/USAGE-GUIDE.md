# 🍎 水果2048 - 使用指南

## 快速开始

### 1. 启动游戏

```bash
cd fruit-2048
python3 -m http.server 8080
```

### 2. 打开浏览器

访问: `http://localhost:8080/index.html`

### 3. 开始游戏

- 使用 **方向键** 移动水果
- 相同的水果合并会升级
- 目标：获得 🍑 桃子

## 游戏规则

| 水果 | 合并需要 | 分数 |
|------|----------|------|
| 🍎 苹果 | 初始 | 2 |
| 🍊 橙子 | 2个苹果 | 4 |
| 🍇 葡萄 | 2个橙子 | 8 |
| 🍓 草莓 | 2个葡萄 | 16 |
| 🍉 西瓜 | 2个草莓 | 32 |
| 🍍 菠萝 | 2个西瓜 | 64 |
| 🥭 芒果 | 2个菠萝 | 128 |
| 🥝 猕猴桃 | 2个芒果 | 256 |
| 🍒 樱桃 | 2个猕猴桃 | 512 |
| 🍑 桃子 | 2个樱桃 | 1024 |

## 操作说明

### 键盘控制
- `↑` `↓` `←` `→` - 移动水果
- `R` - 重新开始
- `Escape` - 关闭说明

### 触摸控制
- 在游戏区域滑动
- 或使用屏幕下方的方向按钮

## 游戏功能

### 按钮功能
- **重新开始** - 开始新游戏
- **游戏说明** - 查看详细规则
- **音效** - 开关游戏音效

### 连击系统
- 连续合并获得连击加成
- 2秒内合并算连击
- 连击越高，加成越多

## 故障排除

### 游戏不显示？

1. 打开诊断页面: `verify.html`
2. 查看检查结果
3. 确认所有项都是 ✅

### 水果不显示？

1. 按F12打开开发者工具
2. 查看Console标签
3. 检查是否有错误信息

### 无法操作？

1. 确认游戏板有焦点
2. 点击游戏区域
3. 尝试使用屏幕按钮
