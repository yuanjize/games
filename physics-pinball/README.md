# 物理弹球游戏 (Physics Pinball)

一个基于原生JavaScript、HTML5 Canvas和CSS3构建的物理引擎弹球游戏，具有真实的物理模拟和精美的视觉效果。

## 游戏特性

### 🎮 核心功能
- **真实物理引擎**：重力、碰撞、反弹、摩擦力模拟
- **多种游戏元素**：反弹器、目标区域、斜坡、挡板
- **分数系统**：不同元素提供不同分数
- **生命系统**：球掉落底部损失生命，生命用尽游戏结束
- **关卡系统**：随着分数增加自动升级关卡
- **本地存储**：最高分自动保存

### 🎨 视觉设计
- **经典弹球台外观**：复古与现代结合的设计风格
- **动态灯光效果**：渐变背景、网格效果、粒子系统
- **3D立体效果**：球体、挡板、反弹器的立体渲染
- **响应式布局**：适配各种屏幕尺寸

### 🎵 音频系统
- **Web Audio API**：实时生成各种音效
- **音效反馈**：碰撞、得分、发射等都有对应音效

### 🎯 控制方式
- **键盘控制**：左右箭头键控制挡板，空格发射球
- **鼠标控制**：移动鼠标控制挡板，点击发射球
- **触摸控制**：触摸设备支持滑动和点击

## 游戏玩法

### 目标
用挡板反弹球，击中各种游戏元素获取分数，获得尽可能高的分数。

### 游戏元素
- **反弹器 (Bumper)**：+100分，提供强力反弹
- **目标区域 (Target)**：+250分，击中后短暂激活
- **斜坡 (Ramp)**：+50分，改变球的方向
- **额外生命**：过关奖励生命，最多5条

### 控制说明
- **← / → 箭头键**：移动左右挡板
- **空格键**：发射球
- **R键**：重新开始游戏
- **P键**：暂停/继续游戏
- **鼠标拖动**：控制挡板位置
- **点击**：发射球

### 游戏规则
1. 球掉落底部会损失一个备用球
2. 备用球用尽会损失一条生命
3. 生命用尽游戏结束
4. 达到一定分数后自动升级关卡
5. 升级关卡会增加生命和游戏难度

## 技术架构

### 核心技术
- **纯原生JavaScript**：无外部库依赖
- **HTML5 Canvas**：2D图形渲染
- **CSS3**：响应式布局和动画
- **Web Audio API**：音频处理

### 物理引擎
- **向量数学**：位置、速度、加速度计算
- **碰撞检测**：圆形-矩形、圆形-圆形碰撞
- **动量守恒**：弹性碰撞计算
- **反弹物理**：基于角度和速度的反弹

### 代码结构
- `index.html`：游戏主页面结构
- `style.css`：样式和动画
- `game.js`：游戏逻辑和物理引擎
