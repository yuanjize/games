# 贪吃蛇游戏

一个使用纯原生JavaScript、HTML和CSS实现的经典贪吃蛇游戏。

## 游戏特色

- **原生技术**：不使用任何外部库或框架
- **响应式设计**：支持桌面和移动设备
- **多种食物类型**：不同颜色食物提供不同分数
- **渐进难度**：随着分数增加，游戏速度逐渐加快
- **视觉反馈**：蛇身渐变颜色、食物特效动画
- **音效系统**：吃食物、移动、游戏结束等音效
- **最高分记录**：使用本地存储保存最高分数

## 游戏玩法

1. 用方向键控制蛇的移动方向
2. 吃到食物可以增加分数和蛇的长度
3. 避免撞到墙壁或自己的身体
4. 游戏会随着分数增加而逐渐加快速度

## 食物类型

| 颜色 | 类型 | 分数 | 效果 |
|------|------|------|------|
| 🔴 | 普通食物 | +10分 | 基础分数 |
| 🟡 | 黄金食物 | +30分 | 高分奖励 |
| 🟢 | 快速食物 | +20分 | 增加游戏速度 |
| 🔵 | 神秘食物 | 随机分数 | 5-45分随机分数 |

## 控制方式

### 桌面设备
- **方向键**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **Enter键**：开始游戏

### 移动设备
- **触摸方向按钮**：控制蛇的移动方向
- **暂停按钮**：暂停游戏
- **开始按钮**：开始游戏

## 技术特点

1. **纯原生实现**：仅使用原生JavaScript、HTML5、CSS3
2. **Canvas渲染**：使用Canvas 2D API进行游戏渲染
3. **响应式布局**：使用flexbox和grid实现适配不同屏幕
4. **游戏状态管理**：清晰的状态机设计
5. **性能优化**：60fps流畅游戏体验
6. **本地存储**：使用localStorage保存游戏记录

## 运行游戏

1. 将`snake-game`文件夹复制到您的服务器
2. 在浏览器中打开`index.html`文件
3. 或者直接双击`index.html`文件
