# 🚀 太空射击游戏 (Space Shooter)

一个使用原生JavaScript开发的跨平台太空射击游戏，支持桌面和移动设备。

## ✨ 游戏特色

- **🎮 双平台控制**：键盘（桌面）和触摸屏（移动设备）完美支持
- **🎯 三种难度**：简单、中等、困难，适合不同水平的玩家
- **💥 华丽特效**：爆炸动画、激光拖尾、星空背景
- **🔊 完整音效**：射击、爆炸、击中、游戏结束音效
- **📊 数据统计**：最高分、生存时间、总游戏次数记录
- **📱 响应式设计**：自适应各种屏幕尺寸

## 🎯 游戏目标

控制你的太空飞船，躲避陨石和敌机，同时发射子弹击毁敌机获得分数。尽可能长时间生存并获得高分！

## 🎮 控制方式

### 桌面设备（键盘）
- **移动飞船**：方向键 ↑ ↓ ← → 或 WASD
- **发射子弹**：空格键
- **暂停/继续**：P 键

### 移动设备（触摸屏）
- **移动飞船**：左侧摇杆控制
- **发射子弹**：右侧发射按钮 或 点击屏幕右侧区域
- **持续射击**：按住发射按钮

## 🎲 游戏规则

1. **初始生命**：3点生命值
2. **得分方式**：
   - 击毁敌机：+10分
   - 击毁陨石：+5分
3. **损失生命**：
   - 被陨石撞击：-1生命
   - 被敌机撞击：-1生命
4. **能量提升**：
   - ❤️ 生命值：恢复1点生命
   - ⚡ 快速射击：5秒内提高射速
   - 🛡️ 护盾：3秒内无敌状态
5. **游戏结束**：生命值降至0

## 🔧 开发指南

### 添加新敌机类型
在 `game.js` 中修改 `spawnEnemy` 方法：
```javascript
// 修改敌机类型生成逻辑
const enemyTypes = ['normal', 'fast', 'heavy']; // 添加新类型
```

### 修改游戏难度
在 `SpaceShooterGame` 构造函数中调整难度设置：
```javascript
this.difficultySettings = {
    easy: { enemySpawnRate: 1.0, enemySpeed: 1.5, asteroidSpawnRate: 0.8 },
    medium: { enemySpawnRate: 1.5, enemySpeed: 2.0, asteroidSpawnRate: 1.2 },
    hard: { enemySpawnRate: 2.0, enemySpeed: 2.5, asteroidSpawnRate: 1.5 }
};
```

### 自定义游戏外观
在 `style.css` 中修改颜色和动画：
```css
.player-ship {
    background: linear-gradient(45deg, #00ffff, #0088ff);
    /* 自定义渐变颜色 */
}
```
