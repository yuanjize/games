# 智能井字棋游戏

一个使用纯原生JavaScript、HTML和CSS开发的井字棋游戏，包含三种不同难度的AI对手。

## 游戏特色

### 🎮 游戏功能
- 经典的3x3井字棋对战
- 玩家（X）对战AI（O）
- 三种AI难度级别
- 实时游戏状态显示
- 走棋历史记录
- 游戏统计跟踪
- 音效和动画效果
- 响应式设计，支持手机和电脑

### 🤖 AI智能级别
1. **简单模式** - AI随机走棋，适合初学者
2. **中等模式** - AI使用基本策略，会尝试阻挡和进攻
3. **困难模式** - AI使用Minimax算法（带Alpha-Beta剪枝），不可战胜的最优策略

### ✨ 视觉效果
- 现代化的UI设计
- 渐变色主题
- 流畅的动画效果
- 获胜线条高亮显示
- 响应式布局适配各种屏幕

## 游戏控制

### 🖱️ 鼠标/触摸控制
- 点击棋盘格子放置"X"
- 点击按钮选择难度级别
- 使用控制按钮进行游戏操作

### ⌨️ 键盘快捷键
- **空格键** - 开始新游戏
- **R键** - 悔棋
- **H键** - 显示提示
- **1键** - 选择简单难度
- **2键** - 选择中等难度
- **3键** - 选择困难难度

## 游戏特性详解

### 🏆 胜负判断
- 横、竖、斜三个相同标记连成一线即获胜
- 棋盘填满而无获胜者为平局
- 获胜时显示动画线条

### 📊 统计系统
- 玩家获胜次数统计
- AI获胜次数统计
- 平局次数统计
- 总游戏局数统计
- 数据通过localStorage本地存储

### 🔄 游戏功能
- **新游戏** - 重置棋盘开始新对局
- **悔棋** - 撤销上一步操作
- **提示** - 显示建议走棋位置
- **音效开关** - 开启/关闭游戏音效
- **难度选择** - 实时切换AI难度
- **重置统计** - 清除所有统计数据

## 技术架构

### 🛠️ 技术栈
- **HTML5** - 游戏结构和内容
- **CSS3** - 样式和动画效果
- **原生JavaScript** - 游戏逻辑和AI算法
- **Web Audio API** - 音效生成
- **SVG** - 获胜线条绘制

### 🧠 AI算法
1. **简单AI** - 随机选择空位
2. **中等AI** - 优先级策略：
   - 检查是否能立即获胜
   - 检查是否需要阻挡玩家获胜
   - 优先选择中心位置
   - 其次选择角落位置
   - 最后随机选择

3. **困难AI** - Minimax算法：
   - 递归搜索所有可能走棋
   - 评估每个位置的分值
   - Alpha-Beta剪枝优化性能
   - 确保最优策略（不可战胜）

### 💾 数据存储
- 使用`localStorage`保存游戏统计数据
- 游戏状态实时更新
- 历史记录缓存最近10步

## 游戏规则

1. 玩家执"X"，AI执"O"
2. 玩家先手，轮流在3x3的棋盘上放置标记
3. 率先将三个相同标记连成一线（横、竖、斜）者获胜
4. 棋盘填满而无获胜者为平局
