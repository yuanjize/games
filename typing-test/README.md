# 打字速度测试游戏

一个使用原生 JavaScript、HTML 和 CSS 实现的打字速度测试游戏，旨在帮助用户提高打字速度和准确性。

## 功能特点

### 核心功能
- 随机文本生成（支持多种难度和语言）
- 实时打字速度检测
- 准确率计算
- 错误字符标记
- 进度跟踪
- 计时功能

### 游戏模式
1. **字数模式** - 完成指定单词数的文本
2. **时间模式** - 在固定时间内尽可能多地打字
   - 1分钟模式
   - 3分钟模式
3. **自定义模式** - 自定义单词数或时间

### 难度等级
- **简单** - 常用词汇，短句
- **中等** - 普通文本，完整句子
- **困难** - 技术文本，专业术语

### 语言支持
- **中文** - 常用中文文本
- **英文** - 英文文本

## 游戏界面

### 主要区域
1. **游戏控制区** - 开始、重启、设置按钮
2. **统计显示区** - 实时显示WPM、准确率、时间等信息
3. **打字区域** - 显示文本供用户打字
4. **历史记录区** - 保存和显示历史成绩
5. **设置面板** - 自定义游戏设置

### 视觉反馈
- **正确字符** - 绿色高亮显示
- **错误字符** - 红色高亮并删除线
- **当前字符** - 蓝色高亮显示
- **跟随光标** - 实时跟随输入位置

## 使用方法

### 开始游戏
1. 点击"开始新游戏"按钮
2. 选择合适的游戏模式和难度
3. 将光标聚焦到输入框开始打字

### 游戏进行中
- 输入框会实时显示您的输入
- 统计信息会实时更新
- 错误字符会被标记出来
- 可以使用退格键删除错误输入

### 游戏结束
- 完成所有文本或时间用完
- 显示最终成绩和评价
- 自动保存到历史记录

### 快捷键
- `空格键` - 开始/暂停游戏
- `Esc键` - 重置游戏
- `Ctrl+N` - 开始新游戏
- `F1键` - 显示帮助

## 统计指标

### WPM（每分钟单词数）
- 基于标准英语单词（5个字符）计算
- 实时更新显示当前速度
- 历史记录中保存最佳成绩

### 准确率
- 计算正确字符占总输入字符的百分比
- 统计不包括空格和标点的准确性

### 错误统计
- 记录总的错误次数
- 实时显示当前错误数量

### 时间统计
- 显示已用时间
- 实时更新计时器
